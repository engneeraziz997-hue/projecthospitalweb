## 📋 نظرة عامة

**مستشفى الشفاء** هو موقع ويب تقديمي (Landing Page) حديث ومتجاوب لمستشفى متعدد التخصصات. يتميز الموقع بتصميم عصري يعتمد على تقنية **Glassmorphism** مع واجهة محادثة ذكية (Chatbot) ونموذج حجز مواعيد تفاعلي.

## ✨ المميزات الرئيسية

### 🎨 التصميم
- **تصميم زجاجي عصري (Glassmorphism)** مع تأثيرات ضبابية
- **تدرجات لونية جذابة** باللونين الأزرق والسماوي
- **رسوم متحركة سلسة** عند التمرير والتفاعل
- **شاشة تحميل أنيقة** مع شعار المستشفى
- **تصميم متجاوب بالكامل** يعمل على جميع الأجهزة

### 🤖 الوكيل الذكي
- **واجهة محادثة تفاعلية** مدمجة في الصفحة الرئيسية
- **تصميم شبيه بتطبيقات المحادثة** (رسائل المستخدم/الروبوت)
- **مؤشر كتابة حي (Typing Indicator)**
- **زر محادثة عائم (Floating Chat Button)**
- **جاهز للربط مع API** للردود الذكية

### 📅 نظام الحجز
- **نموذج حجز مواعيد** سهل الاستخدام
- **اختيار القسم الطبي** من قائمة منسدلة
- **إدخال البيانات الشخصية** (الاسم، الهاتف، التاريخ)
- **زر تأكيد الحجز** مع تأثيرات تفاعلية

### 📊 الأقسام والمحتوى
- **قسم "عن المستشفى"** مع إحصائيات وإنجازات
- **عرض الأقسام الطبية** (6 تخصصات رئيسية)
- **عرض فريق الأطباء** مع تخصصاتهم
- **عرض المرافق الطبية** والخدمات
- **إحصائيات حية** مع عدادات متحركة

### 🔧 التفاعليات
- **شريط تنقل ثابت** يتغير عند التمرير
- **قائمة متنقلة (Hamburger Menu)** للهواتف
- **زر العودة للأعلى** يظهر عند التمرير
- **عدادات إحصائية متحركة** عند الوصول للقسم
- **رسوم متحركة عند التمرير (Scroll Animations)**

## 🛠️ التقنيات المستخدمة

| التقنية | الاستخدام |
|---------|-----------|
| **HTML5** | هيكل الصفحة والعناصر |
| **CSS3** | التنسيقات، التدرجات، الرسوم المتحركة |
| **JavaScript** | التفاعليات، المحادثة، العدادات |
| **Font Awesome 6** | الأيقونات الطبية |
| **Google Fonts (Cairo)** | الخط العربي |
| **Intersection Observer API** | الرسوم المتحركة عند التمرير |

## 🚀 التثبيت والتشغيل

### 1. استنساخ المستودع
```bash
git clone https://github.com/username/alshafa-hospital.git
cd alshafa-hospital
```

### 2. التشغيل المحلي
```bash
# باستخدام Python
python -m http.server 8000

# أو باستخدام Node.js
npx serve

# أو باستخدام PHP
php -S localhost:8000
```

### 3. فتح الموقع
```
http://localhost:8000
```

> **ملاحظة:** الموقع يعمل بالكامل على المتصفح بدون حاجة لخادم خلفي.
